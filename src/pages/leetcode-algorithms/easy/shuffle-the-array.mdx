Given the array `nums` consisting of `2n` elements in the form `[x1,x2,...,xn,y1,y2,...,yn]`.
Return the array in the form of `[x1,y1,x2,y2,...,xn,yn]`.

### Situation
- The length of array must be an even number `len(arr) == 2n`

#### Example 1
```
Input: nums = [2,5,1,3,4,7], n = 3
Output: [2,3,5,4,1,7] 
Explanation: Since x1=2, x2=5, x3=1, y1=3, y2=4, y3=7 then the answer is [2,3,5,4,1,7].
```
#### Example 2
```
Input: nums = [1,2,3,4,4,3,2,1], n = 4
Output: [1,4,2,3,3,2,4,1]
```
#### Example 3
```
Input: nums = [1,1,2,2], n = 2
Output: [1,2,1,2]
```

### Solution
```python copy filename='shuffle_the_array.py'
def array_shuffle(arr: List[int], n: int):
    new_arr = []
    for i in range(n):
        new_arr.append(arr[:n][i])
        new_arr.append(arr[n:][i])

    return new_arr


print(array_shuffle([1, 3, 5, 2, 4, 6], 3)) # [1,2,3,4,5,6]
```
